# Native Accessibility Checks

This page details the complete set of accessibility checks that AMA automatically runs on your app to help identify and resolve common issues.

## Pressable element checks

- [x] Missing accessibility role
- [x] Missing accessibility label
- [x] Contrast ratio too low
- [x] Tappable area too small
- [x] Accessibility label in all caps
- [x] All caps text

## Text element checks

- [x] Contrast ratio too low
- [x] All caps text
- [x] No headings used

## UI Interactions

- [x] No accessibility state handling
    - [x] No aria-checked handling
    - [x] No aria-expanded handling
    - [x] No aria-selected handling
    - [x] No aria-disabled handling
    - [x] No aria-busy handling

- [ ] Focus trapped in form fields
